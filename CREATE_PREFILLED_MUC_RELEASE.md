# Create a prefilled MUC Release

Click the link below. GitHub will open the **New Release** form with the complete
MUC text already inserted.

You only need to:

1. replace `vX.Y.Z` in the tag, title, filenames, and `/src` link;
2. replace `MOD_THE_SIMS_URL`;
3. replace `SIMFILESHARE_URL`;
4. write the changelog manually;
5. attach the PLAYER and MODDER ZIP files;
6. review and publish the Release.
