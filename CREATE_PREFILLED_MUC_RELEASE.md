Before publishing:

1. Replace `vX.Y.Z` in the **tag** and **Release title**.
2. Replace `MOD_THE_SIMS_URL` once.
3. Fill in the changelog sections: Added, Changed, Fixed, and Removed.
4. Delete any unused changelog section.
5. Attach the PLAYER and MODDER ZIP archives.
6. Review the Release and publish it.

The filenames shown in the template intentionally end with `vX`. They are only examples explaining the archive structure, so you do not need to replace the version throughout the Release text.

The source-code section contains instructions and a URL format only. You do not need to replace a source placeholder in the Release body.
