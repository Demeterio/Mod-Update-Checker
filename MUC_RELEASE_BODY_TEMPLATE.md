## Which ZIP should I download?

### Players

Download:

`Demeterio_ModUpdateChecker_PLAYER_vX.Y.Z.zip`

Use this archive when you only want to install and use Mod Update Checker in
The Sims 4.

```text
Demeterio_ModUpdateChecker_PLAYER_vX.Y.Z.zip
├─ Demeterio_ModUpdateChecker_Script_vX.Y.Z.ts4script
├─ Demeterio_ModUpdateChecker_Generic_vX.Y.Z.package
├─ Demeterio_ModUpdateChecker_Installation_vX.Y.Z.txt
└─ Demeterio_ModUpdateChecker_Compatibility_vX.Y.Z.txt
```

Installation instructions and compatibility information are included inside
the ZIP.

### Mod creators

Download:

`Demeterio_ModUpdateChecker_MODDER_vX.Y.Z.zip`

Use this archive only when you want to add MUC support to your own mod or test
a MUC declaration package.

```text
Demeterio_ModUpdateChecker_MODDER_vX.Y.Z.zip
├─ Demeterio_ModUpdateChecker_Script_vX.Y.Z.ts4script
├─ Demeterio_ModUpdateChecker_Generic_vX.Y.Z.package
├─ Demeterio_ModUpdateChecker_Installation_vX.Y.Z.txt
├─ Demeterio_ModUpdateChecker_Compatibility_vX.Y.Z.txt
├─ Demeterio_ModUpdateChecker_Declaration_Template_vX.Y.Z.package
└─ Documentation/
   └─ Demeterio_ModUpdateChecker_Modder_Guide_vX.Y.Z.pdf
```

The MODDER ZIP includes the declaration template and the complete modder guide.

Do not install or distribute the unchanged declaration template. Copy it into
your own project and follow the Modder Guide.
